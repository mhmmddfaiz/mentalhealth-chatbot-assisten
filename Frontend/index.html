Arsitektur Sistem

Dibangun di atas IndoBERT
yang dioptimasi untuk kesehatan mental

Sistem ini menggabungkan kekuatan pre-trained language model Bahasa Indonesia dengan fine-tuning domain spesifik untuk memahami nuansa emosional.

Frontend — Web Interface

Antarmuka percakapan responsif yang dibangun dengan HTML/CSS/JavaScript. Menampilkan hasil analisis emosi secara real-time kepada pengguna.

React.js · Tailwind

Keamanan & Privasi

Enkripsi end-to-end untuk semua percakapan. Data sensitif kesehatan mental pengguna tidak pernah dibagikan tanpa persetujuan eksplisit.

Komparasi Model — Backend

Komparasi Fine-Tuning IndoBERT, mBERT, dan XLM-RoBERTa pada Klasifikasi Emosi pada Chatbot Kesehatan Mental IndoBER · mBERT · XLM-RoBERTa
